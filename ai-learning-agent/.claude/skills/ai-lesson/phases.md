# Phases & Lessons (可扩展)

## P0 人–AI 协作心智模型与边界
- L1: LLM 为什么会“看似理解但会错”：概率生成、上下文窗口、不可见世界状态
- L2: 可验证协作：输出契约、引用、验收标准、回滚
- L3: 风险分级：事实/概念/推理/方案/价值判断的差异

## P1 问题拆解与输入规格
- L1: 目标-约束-假设-验收（G/C/A/AC）
- L2: 输入/输出 Schema 与稳定接口
- L3: 评估集与反例驱动

## P2 Prompt 工程
- L1: 指令层级与冲突处理；Few-shot；反幻觉策略
- L2: 工具调用与输出结构化（JSON/YAML）
- L3: 自检清单与失败模式

## P3 Context / RAG
- L1: 检索策略（BM25/向量/混合）与 Chunking
- L2: 引用与证据链；RAG 评估（Recall/Precision）
- L3: 注入动态上下文与安全（prompt injection）

## P4 Agent / MCP
- L1: 规划-执行-反思；状态机；工具编排
- L2: MCP 基础：工具注册、权限、沙箱、观测
- L3: 回滚与审计日志

## P5 复利系统（资产化）
- L1: Prompt 资产库结构；命名；版本化
- L2: 评估集与回归测试
- L3: 周复盘：知识索引、复习队列、能力图谱
