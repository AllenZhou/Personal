# AI Learning Agent (Claude Code Skills)

This repo scaffolds a learning agent using Claude Code Skills.

## Quick start
1) Install Claude Code (see Anthropic docs)
2) Install Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3) From repo root:
   - bash .claude/skills/ai-lesson/scripts/journal_init.sh
4) Run Claude Code and invoke skills:
   - /ai-lesson
   - /ai-status
   - /ai-review

## Universal Learning System

For learning any domain, use the universal lesson system:
- `/universal-lesson [domain] [topic-or-phase]`
- See `UNIVERSAL_SYSTEM.md` for details
