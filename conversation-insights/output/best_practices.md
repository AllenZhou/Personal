## 工具使用技巧

### TT-001: 并行搜索
**场景**：需要在多个位置查找信息
**做法**：一次发送多个 Grep/Glob 请求，而不是逐个发送
**效果**：减少等待时间

### TT-002: 使用 Task 工具分解复杂任务
**场景**：多步骤复杂任务
**做法**：使用 Task 工具派发子任务给专门的 agent
**效果**：保持上下文清晰，提高处理质量
