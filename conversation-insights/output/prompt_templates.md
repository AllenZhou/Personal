# Prompt 模板库

基于 89 个高质量对话（score >= 70, resolved, corrections <= 1）提取的有效模式。

---

## 1. 研究调研类 (28 个高质量样本)

### 模板 A：角色扮演 + 背景设定
```
现在你是一个{领域}专家，你了解{相关知识领域}。
{具体情境描述}
你帮我{分析/推测/评估}{具体问题}。
```

**示例**：
> 现在你是一个中东的国际问题专家，然后你了解美国和以色列和伊朗的所有的情况。今天早上美国袭击了伊朗的核设施，你帮我推测一下伊朗后续的行动或者是反应。

### 模板 B：结构化角色定义
```
- Role: {角色名称}
- Background: {用户背景和目标}
- Profile: {角色能力描述}
- Task: {具体任务}
```

**示例**：
> - Role: 国际贸易分析师和商业策略规划师
> - Background: 用户希望借助迪拜的地理优势和DMCC平台...
> - Profile: 你是一位经验丰富的...

## 2. 写作类 (21 个高质量样本)

### 模板 C：结合文件 + 明确输出
```
结合{文件/资料}和{参考标准}，
我需要{输出类型}，
说明{具体内容和目的}。
```

**示例**：
> 结合这个文件和国务院颁发的《关于进一步完善海外综合服务体系的指导意见》，我需要给中国政府出一个报告，说明盈天国际有能力在中东承接综合服务体系的工作。

### 模板 D：要点列举法
```
我需要一份{文档类型}，用于{目的}，
{文档}中要包含几个要点：
1. {要点1}
2. {要点2}
3. {要点3}
...
```

**示例**：
> 我要委托中国供应商进行产品研发和供应，我现在需要一份保密协议，协议中要包含几个要点：1 保护范围 2 方法 3 违约责任 4 适用法律

## 3. 头脑风暴类 (14 个高质量样本)

### 模板 E：角色扮演 + 开放讨论
```
标题："扮演{角色}"
我将扮演一个{角色}的角色。
如果您需要帮助{任务领域}，欢迎向我提出问题。
我将分享有关{具体方面}等方面的指导。
```

### 模板 F：约束条件 + 迭代优化
```
现在你是一个{专家角色}，
我需要你{任务}，
{任务}要{约束1}，
{任务}要{约束2}。
```

**示例**：
> 现在你是一个产品营销专家，我需要你给一个产品命名，产品的命名要体现连接人，连接产品，连接业务的含义。产品命名要简单好记忆。

## 4. 学习类 (11 个高质量样本)

### 模板 G：具体技术任务
```
Create/实现 {具体技术任务}。
{具体要求和约束}。
Name/命名为 {变量名}。
```

## 最佳实践总结

### 高分 Prompt 的共同特点

| 特点 | 说明 |
|------|------|
| **角色设定** | 明确 AI 扮演的专家角色 |
| **背景信息** | 提供足够的上下文 |
| **具体任务** | 清晰说明期望输出 |
| **约束条件** | 列出具体要求和限制 |
| **结构化** | 使用编号、要点、分段 |

### 低分 Prompt 的常见问题

| 问题 | 示例 |
|------|------|
| 过于简短 | "帮我写个文案" |
| 缺少上下文 | "分析一下这个" |
| 目标模糊 | "优化一下" |
| 无约束条件 | "随便写个方案" |

---

## 使用建议

1. **复杂任务**：使用模板 B（结构化角色定义）
2. **文档生成**：使用模板 D（要点列举法）
3. **创意任务**：使用模板 E/F（角色扮演 + 约束）
4. **技术任务**：使用模板 G（具体要求）
